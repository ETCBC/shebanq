Clause kind clause_kindÂ¶

frequency table of values

This feature is present on objects of type clause.

Caution

In version 4b this feature is called kind.

This feature divides the clauses into three types: verbal, nominal and without predication. It is related to the feature typ on clauses, in the sense that each of the values of clause_kind corresponnds to a set of values of typ.

So, this is essentially a feature for convenience: it leads to more concise queries of which the intention is also clearer.
