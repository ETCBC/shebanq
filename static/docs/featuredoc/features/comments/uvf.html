Univalent final consonant -consonantal-transliterated uvfÂ¶

frequency table of values

The consonantal representation of the univalent final consonant of a word ETCBC transliteration.

This feature is present on objects of type word.

For more about features on morphemes, see Word grammar.

Caution

The meanings of the univalent final forms need to be listed.

The univalent final consonant act as markers for functions which do not have contrasting values, e.g. a locative.

Note

The value absent indicates absence (there is no empty value).
